# 数据结构

## 前缀和

```cpp
vector<int> nums;
vector<long long> pre(nums.size() + 1);

// one line 版本
std::partial_sum(nums.begin(), nums.end(), pre.begin() + 1);

// 循环 版本
for (int i = 0; i < n; i++) {
	pre[i + 1] = pre[i] + nums[i];
}

/// 查询左闭右开 [l, r) 区间的和
long long sum = pre[r] - pre[l];
```

## 差分

差分与前缀和“互为逆运算”：差分数组的前缀和还原原数组。

```cpp
vector<int> nums;
int n = nums.size();
vector<long long> diff(n + 1);

// 给左闭右开区间 [l, r) 都加上 v
auto add = [&] (int l, int r, int v) {
	diff[l] += v;
	diff[r] -= v;
};

// 前缀和重建原数组
auto build = [&] () {
	for (int i = 1; i < n; i++)
		diff[i] += diff[i - 1];
	diff.resize(n);
};
```

# 图论

## 最短路

### Dijkstra

适用：非负边权的单源最短路。复杂度 `O((n + m) log n)`。

```cpp
// {x, weight}
vector<vector<pair<int, int>>> g;
int n = g.size();

auto dijkstra = [&] (int start) -> vector<long long> {
	vector<long long> dist(n, LLONG_MAX / 2);
	// {dist[x], x}
	priority_queue<pair<long long, int>, vector<pair<long long, int>>, greater<>> pq;
	
	dist[start] = 0;
	pq.emplace(0, start);
	while (!pq.empty()) {
		auto [d, x] = pq.top();
		pq.pop();
	
		if (d != dist[x]) continue;
		
		for (auto [y, w] : g[x]) {
			if (dist[y] > d + w) {
				dist[y] = d + w;
				pq.emplace(dist[y], y);
			}
		}
	}
	return dist;
};

```

### Floyd

适用：多源最短路，节点数较小。复杂度 `O(n^3)` (n <= 300)
`dist[i][j]` 为 i 到 j 最短距离
Floyd 的 k 是“当前新增允许的中转点”，所以必须最外层枚举。

```cpp
int n;
vector<vector<int>> edges; // u, v, w

vector dist(n, vector<long long>(n, LLONG_MAX / 2));

void floyd() {
	// dist[i][i] = 0
	for (int i = 0; i < n; i++) {
		dist[i][i] = 0;
	}
	
	// 初始化边
	for (auto & e : edges) {
		int u = e[0], v = e[1], w = e[2];
		dist[u][v] = w;
		dist[v][u] = w;
	}

	// 遍历 k -> update i, j
	for (int k = 0; k < n; k++) {
	    for (int i = 0; i < n; i++) {
	        for (int j = 0; j < n; j++) {
	            dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j]);
	        }
	    }
	}
}
```



# 数学
## 素数

### 朴素判断

适用：只判断少量数字是否为素数。单个数复杂度 `O(sqrt(x))`。

```cpp
bool isPrime(int x) {
    if (x < 2) return false;

    for (int i = 2; i * i <= x; i++) {
        if (x % i == 0) {
            return false;
        }
    }

    return true;
}
```

### 埃氏筛

适用：预处理 `[1, n]` 中哪些数是质数。

```cpp
const int MX = 1'000'010;
vector<bool> is_prime(MX, true);

auto init = []() {
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i * i < MX; i++) {
        if (is_prime[i]) {
            for (int j = i * i; j < MX; j += i) {
                is_prime[j] = false;
            }
        }
    }
    return 0;
}();
```

## 质因数分解

### 朴素分解

适用：只分解少量数字，或者数据范围不大时。单个数复杂度 `O(sqrt(x))`。

只保留不同质因数：

```cpp
vector<int> primeFactors(int x) {
    vector<int> res;

    for (int i = 2; i * i <= x; i++) {
        if (x % i == 0) {
            res.push_back(i);
            while (x % i == 0) {
                x /= i;
            }
        }
    }

    if (x > 1) {
        res.push_back(x);
    }

    return res;
}
```

说明：循环结束后如果 `x > 1`，剩下的 `x` 一定是质数。

### SPF

`SPF = Smallest Prime Factor`，最小质因子。
如果 `x >= 2 && spf[x] == x`，说明 `x` 是质数。

适用：需要反复分解很多数字的质因数。预处理复杂度近似 `O(M log log M)`，之后分解一个数接近 `O(质因数个数)`。

```cpp
const int MX = 1'000'010;
vector<int> spf(MX);

auto init = []() {
    for (int i = 0; i < MX; i++) {
        spf[i] = i;
    }

    for (int i = 2; i * i < MX; i++) {
        if (spf[i] == i) {
            for (int j = i * i; j < MX; j += i) {
                if (spf[j] == j) {
                    spf[j] = i;
                }
            }
        }
    }
    return 0;
}();

bool isPrime(int x) {
    return x >= 2 && spf[x] == x;
}

vector<int> primeFactors(int x) {
    vector<int> res;
    while (x > 1) {
        int p = spf[x];
        res.push_back(p);

        while (x % p == 0) {
            x /= p;
        }
    }
    return res;
}
```

# 动态规划 DP

## LIS
找到最长严格递增的子序列的长度
`O(n^2)`
**状态转移方程**：`dp[i] = max(dp[i], dp[j] + 1)`，其中 j < i 且 `nums[j] < nums[i]`

```cpp
int lengthOfLIS(vector<int> &nums) {
	int n = nums.size();
	vector<int> dp(n, 1);
	
	for (int i = 0; i < n; i++) {
		for (int j = 0; j < i; j++) {
			if (nums[j] < nums[i])
				dp[i] = max(dp[i], dp[j] + 1);
		}
	}
	
	return ranges::max(dp);
}
```

`O(nlog n)`
维护一个 `g` 数组，`g[i]` 存储所有长度为 `i+1` 的递增子序列中最小的尾部元素
严格递增时，相等不能扩展长度，所以要用 lower_bound 把相等位置替换掉。
最长不递减子序列，可以用 upper_bound

```cpp
int lis(vector<int>& nums) {
    int n = nums.size();
    vector<int> dp(n);
    vector<int> g;

    for (int i = 0; i < n; i++) {
        int x = nums[i];
        auto it = ranges::lower_bound(g, x);

        dp[i] = it - g.begin() + 1; // 以 i 结尾的 LIS 长度

        if (it == g.end())
            g.push_back(x);
        else
            *it = x;
    }
    return g.size();
}
```

如果只需要 LIS 总长度，答案是 `ranges::max(dp)`，也等于扫描结束后的 `g.size()`。

## 数位 DP

适用：统计 `[1, n]` 中满足某种数位性质的数字个数。

```cpp
class Solution {
public:
    int rotatedDigits(int m) {
        string s = to_string(m);
        int n = s.size();

        vector dp(n, vector(2, -1));
        auto dfs = [&] (this auto &&self, int i, bool has_diff, bool is_limit, bool is_num) -> int {
            if (i == n)
                return is_num && has_diff;

            if (!is_limit && is_num && dp[i][has_diff] != -1) {
                return dp[i][has_diff];
            }

            int ans = 0;
            if (!is_num) // 跳过这位不填数字
                ans += self(i + 1, false, false, false);

            int up = is_limit ? s[i] - '0' : 9; // 是limit, 这位最高填s[i] - '0', 否则填9
            int low = is_num ? 0 : 1;           // 是is_num, 这位填0, 否则只能填1(不然全是前导0了)

            // 枚举第i位填d
            for (int d = low; d <= up; d++) {
                if (d == 3 || d == 4 || d == 7) continue;
                bool ndiff = has_diff || d == 2 || d == 5 || d == 6 || d == 9;
                ans += self(i + 1, ndiff, is_limit && d == up, true);
            }

            if (!is_limit && is_num)
                dp[i][has_diff] = ans;

            return ans;
        };

        // 初始化 is_limit = true, 受到约束, 第一位最多填s[0]
        // 初始化 is_num = false, 受到约束, 第一位不能填0
        return dfs(0, false, true, false);
    }
};
```

参数：
- `i`：当前填到第几位
- `state`：前面数位留下的信息
- `isLimit`：是否贴着上界
- `isNum`：是否已经开始填数字

例子：
- `804`：`state = hasDiff`，不能含 `3/4/7`，至少含一个 `2/5/6/9`
