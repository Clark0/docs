# 离散化

当只关心数据大小关系的时候，用排名代替原数组。

处理离线数列。当原数列值域很大，有负数的情况，一些数据结构(BIT)无法运作。

排序 + 去重 + 二分
```cpp
vector<int> preprocess(const vector<int> &a) {
    int n = a.size();
    vector<int> b(a.begin(), a.end());
    sort(b.begin(), b.end());
    b.erase(std::unique(b.begin(), b.end()), b.end());

    vector<int> rank(a.size());
    for (int i = 0; i < n; i++) {
        rank[i] = lower_bound(b.begin(), b.end(), a[i]) - b.begin() + 1;
    }
    return rank;
}
```

set去重 + 排序, 用哈希表维护离散化映射。
```cpp
vector<int> preprocess(const vector<int> &a) {
    set<int> st(a.begin(), a.end());
    unordered_map<int, int> mp;
    int idx = 1;
    for (int x : st) {
        mp[x] = idx++;
    }

    int n = a.size();
    vector<int> rank(n);
    for (int i = 0; i < n; i++) {
        rank[i] = mp[a[i]];
    }
    return rank;
}
```
