整数二分
```cpp
找到数组中第一个值 >= x 的下标位置
auto it = ranges::lower_bound(nums, x);

找到数组中第一个值 > x 的下标位置
auto it = ranges::upper_bound(nums, x);
```
区间 `[lower_bound(x), upper_bound(x))` 就是所有等于x的元素范围。


寻找满足条件的，最小的位置
```cpp
while (l < r) {
    int mid = (l + r) / 2;
    if (check(mid)) {
        r = mid;
    } else {
        l = mid + 1;
    }
}
return a[l];
```

寻找满足条件，最大的位置
```cpp
while (l < r) {
    int mid = (l + r + 1) / 2;
    if (check(mid)) {
        l = mid;
    } else {
        r = mid - 1;
    }
}
return a[l];
```
