# 树状数组

时间复杂度$O(logN)$ 支持
* 单点修改
* 区间查询

BIT中节点$i$的父节点为 $i + lowbit(i)$;
节点i维护[i - lowbit(i) + 1, i]的区间和


## 单点修改，区间查询
树状数组的下标从1开始，板子里的都是树状数组的下标

https://leetcode.com/problems/range-sum-query-mutable/

```cpp
template<typename T>
struct BIT {
    int n;
    vector<T> t;

    BIT(int s) : n(s), t(s) {}

    void add(int x, T v) {
        assert(0 < x && x < n);
        for (; x < t.size(); x += x & -x)
            t[x] += v;
    }

    T ask(int x) {
        assert(0 <= x && x < n);
        T ans = 0;
        for (; x > 0; x -= x & -x) {
            ans += t[x];
        }
        return ans;
    }
};
```

## 区间修改，单点查询
树状数组维护差分数组。nums下标从1开始，$nums[0] = 0$
$$
diff[i] = nums[i] - nums[i - 1];
$$
先构造差分数组，之后用差分数组建树

```cpp
void upddate(int l, int r, T k) {
    add(l, k);
    add(r + 1, -k);
}
```

## 区间修改，区间查询
log(n)实现区间修改，区间查询。
数组前缀和
$$
a[1] + a[2] ... a[i]\\
= diff[1] * i + diff[2] * (i - 1) ... diff[i] * 1 \\
= (i + 1) * (d[1] + d[2] ... + d[i]) - (d[1] * 1 + d[2] * 2 ... d[i] * i);
$$
额外维护一个数组 $d[i] * i$的值即可。

```cpp
template<typename T>
struct BIT {
    vector<T> t, w;

    BIT(int n) : t(n + 1),  w(n + 1) {}

    void add(int x, T v) {
        for (; x < t.size(); x += (x & -x)) {
            t[x] += v;
            w[x] += v * i;
        }
    }

    /// 区间 +x
    void update(int l, int r, T x) {
        add(l, x);
        add(r + 1, -x);
    }

    /// 计算前缀和
    T ask(int x) {
        T ans = 0;
        for (; x; x &= (x - 1)) {
            ans += t[x] * (x + 1) - w[x];
        }
        return ans;
    }
};
```
