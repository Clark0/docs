> 背包问题是一类经典的动态规划问题，核心特点是「选择是否放入物品」来最大化某种价值或最小化某种代价。

## 0-1 背包

> [!info] 问题描述
> 有 $n$ 个物品，第 $i$ 个物品体积为 $w[i]$，价值为 $v[i]$。
> **每个物品最多选一个**，求体积不超过 capacity 时的最大价值和。

### 核心思路

**第 $i$ 个物品选或不选**：
- 不选：继承前 $i-1$ 个物品的结果
- 选：从前 $i-1$ 个物品剩余容量减去 $w[i]$，加上 $v[i]$

子问题：在剩余容量为 $c$ 时，从前 $i$ 个物品中得到的最大价值和

$$
dfs(i, c) = \max(dfs(i - 1, c), dfs(i - 1, c - w[i]) + v[i])
$$

相关例题：[[494. 目标和]]

### 二维 DP 写法

```cpp
vector<int> w;
vector<int> v;
int capacity;

int zero_one_knapsack(int capacity, vector<int> &w, vector<int> &v) {
    int n = w.size();
    // dp[i][j]：前 i 个物品，容量 j 时的最大价值
    vector<vector<int>> dp(n + 1, vector<int>(capacity + 1, 0));
    
    for (int i = 0; i < n; i++) {
        for (int j = 0; j <= capacity; j++) {
            // 不选第 i 个物品
            dp[i + 1][j] = dp[i][j];
            
            // 选第 i 个物品（如果容量足够）
            if (j >= w[i])
                dp[i + 1][j] = max(dp[i][j], dp[i][j - w[i]] + v[i]);
        }
    }
    
    return dp[n][capacity];
}
```

### 一维空间优化

> [!tip] 倒序遍历
> 0-1 背包优化到一维数组时，**必须倒序遍历容量**，避免重复选择同一物品！

```cpp
int zero_one_knapsack(int capacity, vector<int> &w, vector<int> &v) {
    int n = w.size();
    vector<int> dp(capacity + 1, 0);
    
    for (int i = 0; i < n; i++) {
        // 倒序遍历，防止同一物品被选多次
        for (int j = capacity; j >= w[i]; j--) {
            dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
        }
    }
    
    return dp[capacity];
}
```

## 完全背包

> [!info] 问题描述
> 有 $n$ 个物品，第 $i$ 个物品体积为 $w[i]$，价值为 $v[i]$。
> **每个物品可以选无限次**，求体积不超过 capacity 时的最大价值和。

### 核心思路

**选或不选**：
- 不选 -> $dfs(i - 1, c)$
- 选 -> $dfs(i, c - w[i]) + v[i]$（继续处理第 $i$ 个物品）

$$
dfs(i, c) = \max(dfs(i - 1, c), dfs(i, c - w[i]) + v[i])
$$

> [!tip] 正序遍历
> 完全背包优化到一维数组时，**正序遍历容量**，可以重复选择同一物品！

```cpp
int unbounded_knapsack(int capacity, vector<int> &w, vector<int> &v) {
    int n = w.size();
    vector<int> dp(capacity + 1, 0);
    
    for (int i = 0; i < n; i++) {
        // 正序遍历，允许同一物品被选多次
        for (int j = w[i]; j <= capacity; j++) {
            dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
        }
    }
    
    return dp[capacity];
}
```

## 多重背包

> [!info] 问题描述
> 有 $n$ 个物品，第 $i$ 个物品体积为 $w[i]$，价值为 $v[i]$，最多选 $s[i]$ 个。
> 求体积不超过 capacity 时的最大价值和。

### 方法一：枚举物品个数

> [!note] 枚举策略
> $k$ 从 1 开始枚举，因为不选的情况已经在 dp 中了。

```cpp
vector<int> w;  // 体积
vector<int> v;  // 价值
vector<int> s;  // 最多选几个

int multiple_knapsack(int capacity, vector<int> &w, vector<int> &v, vector<int> &s) {
    int n = w.size();
    vector<int> dp(capacity + 1, 0);
    
    for (int i = 0; i < n; i++) {
        // 倒序遍历，防止同一物品被选超过 s[i] 次
        for (int j = capacity; j >= 0; j--) {
            // 枚举选 1 到 s[i] 个
            for (int k = 1; k <= s[i] && k * w[i] <= j; k++) {
                dp[j] = max(dp[j], dp[j - k * w[i]] + k * v[i]);
            }
        }
    }
    
    return dp[capacity];
}
```

### 方法二：二进制打包优化

> [!tip] 二进制原理
> 选取 $1, 2, 4, 8 ... 2^n$ 相加可以表示出所有小于 $2^{n + 1}$ 的数。
> 例如 $13 = 1 + 4 + 8$，可以表示 1~13 的所有数。
> 
> 因此将每个物品按 $1, 2, 4 ...$ 件打包，转化为 0-1 背包问题。

```cpp
vector<int> w, v, s;
/// 打包后的物品价值和体积
vector<int> pw, pc;

void binary_packing(int n) {
    for (int i = 0; i < n; i++) {
        // 打包 1, 2, 4, 8... 个物品
        for (int j = 1; j <= s[i]; j <<= 1) {
            s[i] -= j;
            pw.push_back(j * v[i]);   // 打包后价值
            pc.push_back(j * w[i]);   // 打包后体积
        }
        // 有剩余，如 13 = 1 + 4 + 8，此时会余下 0
        if (s[i] > 0) {
            pw.push_back(s[i] * v[i]);
            pc.push_back(s[i] * w[i]);
        }
    }
}
```

然后对 `pc`, `pw` 调用 0-1 背包即可。

### 方法三：单调队列优化（选做）

时间复杂度优化到 $O(nW)$，适用于数据范围较大的情况。

## 分组背包

> [!info] 问题描述
> 有 $n$ 组物品，每组中有若干个物品。
> **同一组内的物品至多/恰好选一个**，求体积不超过 capacity 时的最大价值和。

模板代码：

```cpp
// groups[k] 是第 k 组的物品列表：(w, v)
vector<vector<pair<int, int>>> groups;

int group_knapsack(int capacity) {
    vector<int> dp(capacity + 1, 0);
    
    for (auto &group : groups) {
        // 倒序遍历容量
        for (int j = capacity; j >= 0; j--) {
            // 枚举组内选哪个物品（或者不选）
            for (auto &[w, v] : group) {
                if (j >= w) {
                    dp[j] = max(dp[j], dp[j - w] + v);
                }
            }
        }
    }
    
    return dp[capacity];
}
```

## 常见变体

> [!question] 常见问法
> 1. 求「最大价值」：标准问法
> 2. 求「恰好装满」的最大/最小价值：初始化时 $dp[0] = 0$，其余为 $-\inf$ 或 $\inf$
> 3. 求「方案数」：转移方程从 $\max$ 改为求和 $\sum$
> 4. 求「至少选/至少装满」：边界条件需要注意
