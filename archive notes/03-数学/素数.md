# 素数筛

## 试除法

朴素判断x是否为素数

```cpp
bool is_prime(int x) {
    if (x < 2)
        return false;
    for (int i = 2; i * i <= x; i++) {
        if (x % i == 0)
            return false;
    }
    return true;
}
```

## 埃氏筛

时间复杂度`O(nlglgn)`, 遍历1 - n, 找到一个素数(is_prime[x] = true)，然后删掉这个素数的倍数。

```cpp
/// 直接预处理 0 - 1E6直接的素数

const int MAXN = 1e6 + 10;
vector<bool> is_prime(MAXN, true);

int init = []() {
    int n = MAXN;
    is_prime[0] = is_prime[1] = false;

    for (int i = 2; i * i <= n; i++) {
        if (is_prime[i])
            for (int j = i * i; j <= n; j += i)
                is_prime[j] = false;
    }
    return 0;
}();
```

# 欧拉筛

```cpp
const int MAXN = 1e6 + 10;
vector<bool> is_prime(MAXN, true);

init init = []() {
	vector<int> primes;
    is_prime[0] = is_prime[1] = false;
    int n = MAXN;

    for (int i = 2; i <= n; i++) {
        if (is_prime[i])
            primes.push_back(i);

        for (int p : primes) {
            if (p * i > n)
                break;
            is_prime[p * i] = false;
            if (i % p == 0)
                break;
        }
    }
    return 0;
}();
```
