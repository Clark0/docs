# 最短路算法

## 单源最短路 SSSP
### Dijkstra

适用于边权为正数的情况。使用```vector<vector<Node, Weight>>```建图

```cpp
int n;
vector<vector<pair<int, int>>> g(n);
vector<int> dist(n, INT_MAX);

auto dij = [&] (int s = 0) {
    fill(dist.begin(), dist.end(), 0x3f3f3f);
    using Dist = pair<int, int>;
    priority_queue<Dist, vector<Dist>, greater<Dist>> pq;
    pq.emplace(s, 0);
    dist[s] = 0;
    vector<bool> vis(n);
    while (!pq.empty()) {
        int x = pq.top().second;
        pq.pop();
        if (vis[x]) continue;
        vis[x] = true;
        for (auto [y, w] : g[x]) {
            if (dist[y] > dist[x] + w) {
                dist[y] = dist[x] + w;
                pq.emplace(dist[y], y);
            }
        }
    }
};
dij();

```

## Bellmen Ford

建一个边图，vector<tuple<int, int, int>> [x, y, w]代表从x到y边权是w;
最短路上的点总个数一定不大于n, 所以把所有边松弛n - 1遍。

```cpp
int n;
vector<tuple<int, int, int>> g;
vector<int> dist(n, 0x3f3f3f);

auto bf = [&](int s = 0) {
    fill(dist.begin(), dist.end(), 0x3f3f3f);
    dist[s] = 0;
    vector<int> backup = dist;
    for (int p = 0; p < n - 1; p++) {
        swap(backup, dist);
        for (auto [x, y, w] : g) {
            dist[y] = min(dist[y], backup[x] + w);
        }
    }
};
```

## Floyd
多源最短路

```cpp
vector<vector<int>> dist;

void floyd(int n, vector<int> &edges) {
    dist.resize(n, vector<int>(n, 0x3f3f3f));
    for (auto &e : edges) {
        int x = e[0], y = e[1], w = e[2];
        dist[x][y] = dist[y][x] = w;
    }

    for (int i = 0; i < n; i++)
        dist[i][i] = 0;

    for (int k = 0; k < n; k++) {
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j]);
            }
        }
    }
}
```
